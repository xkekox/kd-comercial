# KD Comercial

Aplicativo web da KD Embalagens para:

- gerar orcamentos
- gerar recibos em PDF
- gerar PDF para fornecedor
- salvar registros comerciais

## Uso local

Abra `index.html` no navegador.

## Publicacao

Este repositorio ja esta preparado para publicar via GitHub Pages.

Depois de criar o repositorio no GitHub e enviar os arquivos:

1. Ative `Settings -> Pages -> Build and deployment -> GitHub Actions`
2. O workflow `.github/workflows/deploy-pages.yml` vai publicar o app

## Arquivos principais

- `index.html`
- `app.js`
- `styles.css`
- `manifest.webmanifest`
- `service-worker.js`
